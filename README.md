# SimpleSorter (极简智能排序)

> 一款轻量级 Minecraft Fabric 客户端模组，为你的背包管理带来丝滑体验。

---

## ✨ 功能一览

### 🔑 一键智能排序

打开任意容器界面（背包、箱子、末影箱等），按下 **R 键** 即可一键整理物品。

- **自动堆叠合并** — 将散落在不同格子里的同类物品优先合并为满组，减少空间浪费
- **分类排序** — 物品按照 Minecraft 创造模式物品栏的类别顺序排列：
  1. 工具与实用品
  2. 战斗用品
  3. 建筑方块
  4. 自然方块
  5. 功能性方块
  6. 红石元件
  7. 食物与饮品
  8. 材料
  9. 刷怪蛋
- **纯客户端操作** — 排序通过模拟玩家鼠标点击完成，与服务器完全兼容，可在任何服务器上安全使用

---

### 🖱️ 鼠标增强操作

#### Shift + 拖拽快速转移

按住 **Shift** 后用鼠标 **左键拖拽** 经过多个格子，所经之处的物品会被一个接一个地快速转移到另一侧容器中。

> 💡 比一个格子一个格子点 Shift+Click 快得多！

#### Shift + 双击批量同类转移

手上 **拿着一个物品** 的同时，对同类物品 **Shift + 双击**，会将该容器内 **所有相同物品** 一次性全部快速转移到另一侧。

> 💡 适合从箱子里一次性取出所有钻石/铁锭等同类资源。

#### Space + 双击一键清空

按住 **空格键** 的同时 **双击** 容器中的任意格子，会将 **该容器内全部物品** 一次性快速转移到另一侧。

> 💡 一键清空整个箱子或整个背包，搬家神器！

---

### ⚙️ 可视化配置界面

按下 **Z + I 键**（默认组合键）即可在游戏内打开图形化设置面板：

- **物品分类排序顺序** — 自定义各物品类别的排列优先级
- **Z 键保护开关** — 可以关闭 Z 键要求，改为直接按 I 键打开设置

如果安装了 **Mod Menu**，也可以从 Mod 列表中直接进入设置界面。

## 📦 安装要求

| 依赖 | 是否必须 | 说明 |
|------|---------|------|
| **Fabric Loader** ≥ 0.16.5 | ✅ 必须 | 模组加载器 |
| **Fabric API** | ✅ 必须 | 基础 API 库 |
| **Fabric Language Kotlin** | ✅ 必须 | Kotlin 运行时支持 |
| **Cloth Config API** | ❌ 已内嵌 | 配置界面框架（已打包在模组内，无需另行安装） |
| **Mod Menu** | ❌ 可选 | 提供从 Mod 列表进入设置的入口 |

|:---:|:---:|

| 1.21.1 | ✅ 支持 |

## 🚀 安装方法

1. 确保已安装 **Fabric Loader** 和 **Fabric API**
2. 下载 **Fabric Language Kotlin** 放入 `mods` 文件夹
3. 将 `simplesorter-fabric-x.x.x-1.0.0.jar` 放入 `mods` 文件夹
4. 启动游戏，享受丝滑整理体验！

## 🎮 快捷键速查

| 快捷键 | 功能 | 使用场景 |
|:---:|------|------|
| **R** | 一键排序整理 | 打开任意容器界面时按下 |

## 📝 技术细节

- **纯客户端模组** — 不修改任何服务端逻辑，通过模拟玩家点击操作实现排序
- **多版本架构** — 采用 `core` + `shared` + `platform` 三层架构，一套核心逻辑同时支持多个 Minecraft 版本
- **模块化鼠标增强** — 每个鼠标操作增强功能都是独立的 `MouseTweakModule`，易于扩展和维护
