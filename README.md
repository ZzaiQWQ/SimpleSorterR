# SimpleSorter (极简智能排序)

> 轻量级 Fabric 客户端背包整理模组。

---

## 功能一览

### 一键智能排序

打开背包或箱子等容器界面，按 **R** 整理物品。

- **自动堆叠合并** — 将散落在不同格子里的同类物品优先合并为满组，减少空间浪费
- **Creative Tab 原生排序** — 物品按照 Minecraft 创造模式标签页顺序排列，自动兼容 Mod 物品
- **自定义排序顺序** — 可在配置文件中调整 `tabOrder` 来改变标签页优先级
- **置顶物品** — 配置 `pinnedItems` 让指定物品永远排在最前面
- **动态容器黑名单** — 默认支持所有存储型容器排序（包括 Mod 容器如 Reinforced Chests），功能性容器（熔炉、工作台等）通过 `blockedContainers` 精准类名屏蔽；也可以打开容器后按 **B 键** 一键加入/移出黑名单
- **配置热更新** — 直接编辑 `simplesorter.json` 后，下次按 R 排序时自动生效，无需重启游戏
- **纯客户端操作** — 排序通过模拟鼠标点击完成，服务端无需安装

### 鼠标增强操作

#### Shift + 拖拽快速转移

按住 **Shift** 后用鼠标 **左键拖拽** 经过多个格子，所经之处的物品会被一个接一个地快速转移到另一侧容器中。

> 比一个格子一个格子 Shift+Click 更省事。

#### Shift + 双击批量同类转移

手上 **拿着一个物品** 的同时，对同类物品 **Shift + 双击**，会将该容器内 **所有相同物品** 一次性全部快速转移到另一侧。

> 适合从箱子里一次性取出所有同类物品。

#### Space + 双击一键清空

按住 **空格键** 的同时 **双击** 容器中的任意格子，会将 **该容器内全部物品** 一次性快速转移到另一侧。

> 适合快速清空箱子。

---

### 格子锁定

按住 **Alt 键** 单击某个格子，即可将其**锁定**，锁定的格子会显示遮罩。

- 锁定的格子在按 R 键排序时**不会被移动**
- 锁定的快捷栏格子**无法通过 Q 键丢弃**物品（防止误丢）
- 锁定状态会**自动保存**，不同存档各自独立
- 再次 Alt 单击可**解除锁定**
- 按住 **Alt + 左键拖动** 可以批量锁定/解锁多个格子，第一个格子的状态决定本次拖动是锁定还是解锁
- 遮罩颜色可通过 `lockOverlayColor` 配置，支持 `#RRGGBB` 和 `#AARRGGBB`

颜色代码使用 **Hex 颜色代码**，支持 `#RRGGBB` 和 `#AARRGGBB`。8 位格式是 `ARGB` 顺序：`AA` 透明度，`RR` 红色，`GG` 绿色，`BB` 蓝色，不是 `RGBA`。

`AA` 常用值：`40` 很透明，`80` 半透明，`C0` 比较不透明，`FF` 完全不透明。示例：`#80FF0000` 半透明红色，`#8000FF00` 半透明绿色，`#800000FF` 半透明蓝色。

> 适合固定常用工具位置，也能减少误丢。

---

### 自动补充 (AutoReplacer)

- **工具断裂自动替换** — 当手持工具耐久耗尽断裂时，自动从背包中取出同种物品或同类工具补充到快捷栏
- **可堆叠物品自动补货** — 当手持的方块/物品用完时（如放置方块），自动从背包中补充相同物品
- 可在设置界面中**开关**此功能

> 建筑和挖矿时不用频繁打开背包找替换物品。

---

### Caps Lock 双击批量丢弃

按住 **Caps Lock** 的同时**双击**某个格子，会将**当前容器内所有相同物品**全部丢出（Q 键丢弃）。

> 用来快速清理大量同类物品。

---

### 可视化配置界面

按下 **Z + I 键**（默认组合键）即可在游戏内打开图形化设置面板：

- **标签页排序顺序 (tabOrder)** — 自定义创造模式标签页的排列优先级
- **置顶物品 (pinnedItems)** — 指定始终排在最前面的物品 ID
- **容器黑名单 (blockedContainers)** — 自定义不参与排序的容器类名；也可以直接打开容器按 **B** 自动写入当前容器类名
- **锁定遮罩颜色 (lockOverlayColor)** — 自定义锁定格子的显示颜色，例如 `#80FF0000`
- **Z 键保护开关** — 可以关闭 Z 键要求，改为直接按 I 键打开设置
- **自动补充开关** — 启用/禁用工具断裂和物品用尽时的自动替换功能
- **中英文自适应** — 界面语言随游戏语言自动切换

如果安装了 **Mod Menu**，也可以从 Mod 列表中直接进入设置界面。

> 高级配置可直接编辑 `config/simplesorter.json`，修改后下次按 R 排序时自动生效。

> ⚠️ `blockedContainers` 使用**精准类名匹配**，不会模糊匹配。不同映射/版本下类名可能是 `GenericContainerScreenHandler` 这种 ScreenHandler 名称，不一定是 `ChestMenu` 这种 Menu 名称；最稳妥的方式是打开目标容器后按 **B**，让模组自动写入正确类名。

## 安装要求

| 依赖 | 是否必须 | 说明 |
|------|---------|------|
| **Fabric Loader** ≥ 0.16.5 | ✅ 必须 | 模组加载器 |
| **Fabric API** | ✅ 必须 | 基础 API 库 |
| **Fabric Language Kotlin** | ✅ 必须 | Kotlin 运行时支持 |
| **Cloth Config API** | ❌ 已内嵌 | 配置界面框架（已打包在模组内，无需另行安装） |
| **Mod Menu** | ❌ 可选 | 提供从 Mod 列表进入设置的入口 |


---

## 安装方法

1. 确保已安装 **Fabric Loader** 和 **Fabric API**
2. 下载 **Fabric Language Kotlin** 放入 `mods` 文件夹
3. 根据游戏版本选择对应的 `simplesorter-fabric-版本号-3.5.0.jar` 放入 `mods` 文件夹
4. 启动游戏

## 快捷键速查

所有快捷键都会显示在游戏按键设置里的 **SimpleSorter** 分类下，可自行修改。

| 快捷键 | 功能 | 使用场景 |
|:---:|------|------|
| **R** | 一键排序整理 | 打开任意容器界面时按下 |
| **Alt + 单击** | 锁定/解锁格子 | 防止该格物品被排序移动 |
| **Alt + 左键拖动** | 批量锁定/解锁格子 | 连续拖过多个格子 |
| **B** | 屏蔽/取消屏蔽当前容器 | 当前容器不想参与 R 键排序时 |
| **Caps Lock + 双击** | 批量丢弃同类物品 | 快速清理大量同类物品 |
| **Shift + 拖拽** | 快速批量转移 | 拖过的格子物品逐个移走 |
| **Shift + 双击** | 同类物品全部转移 | 手持物品时对同类 Shift 双击 |
| **Space + 双击** | 一键清空容器 | 整箱搬运 |
| **Z + I** | 打开设置界面 | 随时可用 |

## 技术细节

- **纯客户端模组** — 不修改任何服务端逻辑，通过模拟玩家点击操作实现排序
- **Creative Tab 原生排序** — 直接读取 Minecraft 创造模式标签页构建排序索引，精准还原游戏原生物品顺序，自动兼容所有 Mod 物品
- **多版本架构** — 采用 `core` + `shared` + `platform` 三层架构，一套核心逻辑同时支持多个 Minecraft 版本
- **动态配置系统** — 支持配置文件热更新，编辑后无需重启游戏
- **精准容器匹配** — `blockedContainers` 按容器类名精确匹配，避免模糊匹配误伤其他容器
- **模块化鼠标增强** — 每个鼠标操作增强功能都是独立的 `MouseTweakModule`，易于扩展和维护
